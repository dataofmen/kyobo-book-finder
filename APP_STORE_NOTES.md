# 앱 스토어 릴리즈 노트

## 버전 1.0.0

### 새로운 기능

교보문고 매장에서 원하는 책을 쉽게 찾아보세요!

**주요 기능:**
• 책 제목으로 빠른 검색
• 매장별 정확한 위치 안내 (관, 섹션 정보)
• 실시간 재고 확인
• 전국 주요 교보문고 매장 지원
• 모바일 최적화 디자인

**사용 방법:**
1. 원하는 매장 선택
2. 책 제목 검색
3. 위치와 재고 즉시 확인

새 버전이 어떨까요? 언제든 피드백을 남겨주세요!
